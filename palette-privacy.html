Palette 개인정보 처리방침

최종 개정일: 2026-05-05

본 방침은 Palette(이하 "본 앱")가 어떤 정보를 수집·이용·보관·제공하는지 설명합니다.

1. 처리하는 개인정보 항목

본 앱은 온디바이스 처리를 원칙으로 합니다. 사용자가 촬영하거나 선택한 사진은 기기 내에서만 색상이 추출되며, 원본 사진은 외부 서버로 업로드되지 않습니다.

자동 수집 (선택형 클라우드 동기화 활성화 시)

  • Firebase 익명 인증 식별자(UID)
  • 추출된 색상값(HEX)·팔레트 이름·저장 일시
  • 사진 원본은 업로드되지 않으며, 색상 정보만 클라우드에 보관

카메라·미디어 접근

  • 사용자가 카메라 또는 갤러리 접근을 명시적으로 허용한 경우에만 사용
  • 캡처된 이미지는 기기 메모리에서 색상 추출 후 즉시 폐기

미수집 정보

  • 이름, 전화번호, 주소, 주민등록번호, 이메일
  • 광고 식별자 (IDFA, AAID)
  • 위치 정보, 연락처

2. 이용 목적

  • 사진 → 5색 팔레트 추출 및 표시
  • 사용자가 명시적으로 활성화한 클라우드 동기화 (Firestore에 색상 정보만 기록)
  • 일·주·월·년 리캡 통계 생성 (로컬 또는 사용자 본인 클라우드 데이터 한정)

3. 공유·소셜 기능

사용자가 인스타그램·스레드 등으로 카드 PNG를 공유하는 행위는 사용자 본인의 결정이며, 본 앱은 이를 자동 업로드하지 않습니다.

4. 보관 및 파기

  • 로컬 저장 데이터: 사용자가 앱 데이터 삭제 또는 앱 삭제 시 즉시 파기
  • 클라우드 색상 데이터: 사용자가 삭제 요청 시 30일 이내 파기

5. 제3자 제공 / 위탁

  • Firebase (Google LLC): 인증·색상 데이터 저장 (사용자가 활성화한 경우에 한함)
  • 광고 네트워크·외부 분석 도구 미사용

6. 사용자 권리

  • 처리 정보 열람·정정·삭제·처리정지 요청
  • 클라우드 동기화 언제든 비활성화
  • 요청 채널: contact@dawith.com (실제 이메일로 교체)

7. 아동 보호

본 앱은 만 14세 미만 아동을 대상으로 하지 않으며, 의도적으로 아동의 개인정보를 수집하지 않습니다.

8. 변경 고지

본 방침이 변경될 경우 앱 내 공지 또는 본 페이지를 통해 최소 7일 전 안내합니다.

9. 연락처

  • 책임자: 다윗 (개인 개발자)
  • 이메일: contact@dawith.com (실제 이메일로 교체)
