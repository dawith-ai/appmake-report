Docu 개인정보 처리방침

최종 개정일: 2026-05-05

본 방침은 Docu(이하 "본 앱")가 어떤 정보를 수집·이용·보관·제공하는지 설명합니다.

1. 처리하는 개인정보 항목

본 앱은 온디바이스 처리를 원칙으로 합니다. 사용자가 선택한 문서 파일(PDF·HWP·DOCX 등)의 내용은 사용자 기기 내에서만 분석·변환되며, 외부 서버로 업로드되지 않습니다.

자동 수집 (선택형 클라우드 동기화 활성화 시)

  • Firebase 익명 인증 식별자(UID)
  • 파일 메타데이터 (파일명, 형식, 용량, 처리 일시)

미수집 정보

  • 이름, 전화번호, 주소, 주민등록번호, 이메일
  • 광고 식별자 (IDFA, AAID)
  • 위치 정보, 연락처, 카메라/마이크
  • 문서 파일의 실제 내용 (외부 서버에 저장하지 않음)

2. 이용 목적

  • 문서 변환·편집 기능 제공
  • 사용자가 명시적으로 활성화한 클라우드 동기화 (Firestore에 메타데이터만 기록)
  • 앱 안정성 모니터링 (충돌 로그)

3. 보관 및 파기

  • 로컬 저장 데이터: 사용자가 앱 데이터 삭제 또는 앱 삭제 시 즉시 파기
  • 클라우드 메타데이터: 사용자가 클라우드 동기화 비활성화 또는 삭제 요청 시 30일 이내 파기

4. 제3자 제공 / 위탁

  • Firebase (Google LLC): 인증·메타데이터 저장 (사용자가 활성화한 경우에 한함)
  • 그 외 제3자 제공·위탁 없음
  • 광고 네트워크·분석 도구 미사용

5. 사용자 권리

  • 처리 정보 열람·정정·삭제·처리정지 요청
  • 클라우드 동기화 언제든 비활성화
  • 요청 채널: contact@dawith.com (예시 — 실제 이메일로 교체)

6. 아동 보호

본 앱은 만 14세 미만 아동을 대상으로 하지 않으며, 의도적으로 아동의 개인정보를 수집하지 않습니다.

7. 변경 고지

본 방침이 변경될 경우 앱 내 공지 또는 본 페이지를 통해 최소 7일 전 안내합니다.

8. 연락처

  • 책임자: 다윗 (개인 개발자)
  • 이메일: contact@dawith.com (실제 이메일로 교체)
  • 본 방침에 대한 문의·신고는 위 채널로 접수합니다.
